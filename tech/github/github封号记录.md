三月初，我的 GitHub 账号毫无征兆地被封禁，虽然仓库还能推拉代码，但公开主页直接显示 404。我使用 GitHub 都是合规的操作，我推测是因为将仓库用作图床触发了风控，但这种突然的封禁实在令人费解

整个申诉过程很难受，强制要求非国内手机号，我只能依赖接码平台验证。最糟糕的是处理速度，前后耗时近一个月直到今天才恢复。我真的服了
