### keyword-only 函数

Python3 有一种只有关键字参数，只能传递关键字参数

- 语法

```python
def kwonly(*, name='spam', age=1):
    ...

kwonly('eggs', 18)

TypeError: kwonly() takes 0 positional arguments but 2 were given
```

> 传入非关键字参数会报错

- 正确的语法

```python
kwonly(name='eggs', age=18)
```
