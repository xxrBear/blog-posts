对于我们程序员每天都用的 linux 终端，你有多少了解？现在我来为你介绍几个最常用到的终端命令行命令

- 返回行首

  ```shell
  Ctrl + A
  ```

- 返回行尾

  ```shell
  Ctrl + E
  ```

- 删除光标前的文本

  ```shell
  Ctrl + U
  ```

- 删除光标后的文本

  ```shell
  Ctrl + K
  ```

- 删除光标前一个单词

  ```shell
  Ctrl + W
  ```

- 清屏

  ```shell
  Ctrl + L
  ```
