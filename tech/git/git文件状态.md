## Git 的文件状态

对于 Git 仓库中的文件，我们可以将他们分为 ` 已追踪(Tracked)`和 `未追踪(Untracked)`两部分

对于 `已追踪`的文件，可以分为三个状态：`已提交(committed)`、`已修改(modified)` 和 `已暂存(staged)`
